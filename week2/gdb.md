# GDB

```bash
# Start
gdb
# breakpoint
b [function name, line number]
# run
r [command-line arguments]
# next
n # or next means step forward one block of code
s # step forward one line of code (or step into the function)
# get the value of the variable
p [variable]
# print out the values of all variables
info locals
# backtrace for recursion to show the series of function calls that got you there
bt
# Quit
q
```
